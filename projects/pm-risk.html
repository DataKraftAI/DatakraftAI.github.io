For Project Managers

Project Manager Risk Copilot

AI-powered risk detection & project health dashboard

Problem

Project managers spend hours gathering updates, spotting risks, and tracking project health. Much of this work is manual and inconsistent, especially across multiple projects.

My Solution

A lightweight AI copilot that ingests project updates (Slack, Jira notes, status reports) and produces:

  • Automated risk detection with owners
  • Traffic-light (RAG) project health summary
  • Suggested next actions

How It Works

  • PM pastes project updates or meeting notes
  • Copilot extracts risks, deadlines, and owners
  • Summarizes overall health (Red/Amber/Green)
  • Generates 2–3 next-step recommendations

Deliverables

  • Risk register draft with owners
  • RAG project health snapshot
  • Next-step recommendations
  • Simple, shareable dashboard view
